# Business ERP System

A full-stack ERP (Enterprise Resource Planning) system built using React, Node.js, and MongoDB.

## 🚀 Features
- User Management (CRUD operations)
- REST API integration
- MongoDB database
- Responsive UI using Tailwind CSS

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Styling: Tailwind CSS

## 📂 Project Structure
frontend/
backend/

## ▶️ How to Run

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm start
